## Getting Started

For detailed installation instructions, select your framework below:

<Tabs items={["Expo", "React Native CLI"]}>
  <Tab value="Expo" label="Expo">

  1. Clone the repository:
  ```bash
  git clone https://github.com/okto-hq/okto-sdkv2-react-native-template-app.git
  cd okto-sdkv2-react-native-template-app
  ```

  2. Install dependencies:
  ```bash
  npm install
  ```

  3. Configure your credentials in the app:
  ```typescript
  // Update the configuration in your app
  const config = {
     environment: "sandbox",
    clientPrivateKey: "YOUR_CLIENT_PRIVATE_KEY",
    clientSWA: "YOUR_CLIENT_SWA",
  };

  const googleConfig = {
    webClientId: "YOUR_GOOGLE_WEB_CLIENT_ID",
    iosClientId: "YOUR_GOOGLE_IOS_CLIENT_ID",
    androidClientId: "YOUR_GOOGLE_ANDROID_CLIENT_ID"
  };
  ```

  <CollapsibleCallout type="warning" title="Essential Setup Required">
    Before you begin, set up your [developer dashboard](https://dashboard.okto.tech) by making sure you have :
    - Created your [Okto Developer Dashboard](/docs/developer-admin-dashboard/dashboard-account) account
    - Get your [Okto credits](/docs/developer-admin-dashboard/okto-faucet) from the dashboard
    - Obtained your [API keys](/docs/developer-admin-dashboard/api-key) from the dashboard
    - Enabled the specific [Chains and Tokens](/docs/developer-admin-dashboard/wallet-control) you plan to use in your application
    - Optional: Enabled [Sponsorship](/docs/developer-admin-dashboard/sponsorship) for the desired chains

    If you need help, reach out to us on our <ExternalLink href="/docs/react-native-sdk/troubleshooting">troubleshooting form</ExternalLink> and we will contact you.
    </CollapsibleCallout>

  - Okto API Keys: You need your `CLIENT_PRIVATE_KEY` and `CLIENT_SWA`. Obtain these from the [Okto Developer Dashboard](https://dashboard.okto.tech/login).
  - Okto Account Setup: Ensure you have sufficient <ExternalLink href="https://dashboard.okto.tech/okto-credits">Okto Credits</ExternalLink>, enable the required <ExternalLink href="https://dashboard.okto.tech/wallet-controls">Chains and Tokens </ExternalLink>, and optionally <ExternalLink href="https://dashboard.okto.tech/paymaster">Activate Sponsorship</ExternalLink>.
  - Google OAuth Credentials (if using Google authentication): Create OAuth 2.0 credentials in the <ExternalLink href="/docs/react-native-sdk/authenticate-users/google-oauth/google-console-setup-web">Google Cloud Console</ExternalLink> to get your `GOOGLE_CLIENT_ID`.
    <Callout title="Note">
      1. Create OAuth 2.0 credentials for each platform:
         - Web Client ID (for Expo Go and web)
         - iOS Client ID
         - Android Client ID
      2. Follow our platform-specific setup guides:
         - [Web Client Setup](/docs/react-native-sdk/authenticate-users/google-oauth/google-console-setup-web)
         - [iOS Setup](/docs/react-native-sdk/authenticate-users/google-oauth/google-console-setup-ios)
         - [Android Setup](/docs/react-native-sdk/authenticate-users/google-oauth/google-console-setup-android)
    </Callout>

  4. Start the development server:
  ```bash
  npx expo start
  ```
