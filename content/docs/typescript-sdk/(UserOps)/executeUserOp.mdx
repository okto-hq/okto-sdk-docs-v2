## Execute UserOp

<sub><i>async</i></sub> `executeUserOp(userOp)` executes a user operation.

### Parameters

| Parameter   | Type                        | Description                            | 
|-------------|-----------------------------|----------------------------------------|
| `userOp`     | [`UserOp`](/docs/typescript-sdk/technical-reference#models) | The user operation to execute | 

### Response

<Callout title="Success Response">

| Field Name | Type | Description |
|------------|------|-------------|
| `result` | `Promise<string>` | Returns the job ID of the executed user operation |

</Callout>
