## Estimate

<sub><i>async</i></sub> `estimate(OktoClient, recipientWalletAddress, networkId, tokenAddress, amount)` estimates a user operation.

### Parameters

| Parameter   | Type                        | Description                            | 
|-------------|-----------------------------|----------------------------------------|
| `oktoClient` | [`OktoClient`](/docs/typescript-sdk/technical-reference#models) | Okto client |
| `recipientWalletAddress` | `string` | The recipient wallet address |
| `networkId` | `number` | The network id of the user operation |
| `tokenAddress` | `string` | The token address of the user operation |
| `amount` | `string` | The amount of the user operation |

### Response

<Callout title="Success Response">

| Field Name | Type                  | Description                                   | 
|------------|-----------------------|-----------------------------------------------|
| `result`   | [`Promise<EstimateOrderPayload>`](/docs/typescript-sdk/technical-reference#models) | Returns the estimate order payload | 

</Callout>
