## Setup Requirements

1. **React Environment**
   - A React project (version 16.8 or higher)
   - Node.js (version 14 or higher)
   - npm or yarn package manager
   - For a complete list of dependencies, check our [`package.json`](https://github.com/okto-hq/okto-sdkv2-react-template-app/blob/main/package.json)

2. **Okto Dashboard Setup**
   - Create an account on the [Okto Dashboard](https://dashboard.okto.tech/)
   - Obtain your API Key
   - Configure your desired chains and tokens
   - Enable sponsorship if needed (required for Solana, optional for other chains)

3. **Google OAuth Configuration**
   - Set up a [Google Console Project](https://console.cloud.google.com) for authentication
   - Create OAuth 2.0 client credentials
   - Follow our guide to [set up Google OAuth](/docs/typescript-sdk/authentication/google-oauth/google-console-setup)

<Callout type="info">
Additional authentication methods like Email OTP and Phone OTP will be available soon.
</Callout>

<Callout title="Environment Configuration">
Your project will use the **Sandbox** environment by default. For production deployment, please refer to our [Developer Dashboard  documentation](/docs/developer-admin-dashboard) for upgrade instructions.
</Callout>

## Next Steps

Once you've completed these prerequisites, proceed to the [Quickstart Guide](/docs/typescript-sdk) to integrate the Okto SDK into your React project.
