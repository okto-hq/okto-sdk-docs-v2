## Create NFT Collection

<sub><i>async</i></sub> `nftCollectionCreation(OktoClient, data)` creates a new NFT collection.

### Parameters

| Parameter   | Type                        | Description                            | 
|-------------|-----------------------------|----------------------------------------|
| `oktoClient` | [`OktoClient`](/docs/typescript-sdk/technical-reference#models) | Okto client |
| `data` | [`NFTCollectionCreationIntentParams`](/docs/typescript-sdk/technical-reference#models) | The data for the NFT collection creation |

### Response

<Callout title="Success Response">

| Field Name | Type                  | Description                                   | 
|------------|-----------------------|-----------------------------------------------|
| `result`   | [`Promise<UserOp>`](/docs/typescript-sdk/technical-reference#models) | Returns a UserOp object for NFT collection creation | 

</Callout>
