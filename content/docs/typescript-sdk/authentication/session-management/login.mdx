## Login using OAuth

<sub><i>async</i></sub> `loginUsingOauth(authData)`<a href="https://github.com/okto-hq/okto-sdk-react/blob/main/src/OktoProvider.tsx#L340" target="_blank" rel="noopener noreferrer" style= {{ textDecoration: "none" }}> <IIcon icon="lucide:external-link" height="17" style={{ display: 'inline-block', verticalAlign: 'middle' }} /></a> logins the user using OAuth.

### Parameters

| Parameter   | Type                        | Description                            | 
|-------------|-----------------------------|----------------------------------------|
| `data`     | `AuthData` | Query being provided fetching the data | 

### Response

<Callout title="Success Response">

| Field Name | Type                  | Description                                   | 
|------------|-----------------------|-----------------------------------------------|
| `result`   | [`Promise<User>`](/docs/typescript-sdk/technical-reference#models) | Returns the newly created user object | 

</Callout>
