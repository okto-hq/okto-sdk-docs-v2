<Callout type="info">
### iOS Support
The Okto SDK for Unity is designed to work out of the box with iOS builds. Unity's standard iOS build process is sufficient - no additional dependencies or special configurations in Xcode are required.

You don't need to:
- Add any frameworks
- Modify Info.plist
- Add any dependencies
- Configure any special permissions
</Callout>

## Getting Started

For detailed installation instructions, select your framework below:

<Tabs items={["Unity"]}>
  <Tab value="Unity" label="Unity">

  <div className="max-w-[450px] my-5 mx-auto">
      <img 
          src="/images/unity-images/unity-template.png" 
          alt="Unity Template App" 
          className="w-full rounded-lg shadow-sm"
      />
  </div>

  1. Clone the repository:
  ```bash
  git clone https://github.com/okto-hq/okto-sdkv2-unity-template-app.git
  ```

  2. Change into the project directory and install dependencies:

<Tabs items={['Google Sign-In', 'Play Services Resolver', 'Newtonsoft.Json']}>
    <Tab value="Google Sign-In">
         https://github.com/googlesamples/google-signin-unity/releases
    </Tab>
    <Tab value="Play Services Resolver">
        https://github.com/googlesamples/unity-jar-resolver
    </Tab>
    <Tab value="Newtonsoft.Json">
       "com.unity.nuget.newtonsoft-json": "3.0.2"
    </Tab>
</Tabs>

  3. Set up environment variables:
    ```csharp title="Login.cs"
    webClientId="Google_Web_OAuth"
    ```

  4. Add your API keys to the `.env` file:
    ```csharp title="OktoClient.cs"
    clientPrivateKey="YOUR_CLIENT_PRIVATE_KEY"
    clientSwa="YOUR_CLIENT_SWA"
    ```

  <CollapsibleCallout type="warning" title="Essential Setup Required">
    Before you begin, set up your [developer dashboard](https://dashboard.okto.tech) by making sure you have :
    - Created your [Okto Developer Dashboard](/docs/developer-admin-dashboard/dashboard-account) account
    - Get your [Okto credits](/docs/developer-admin-dashboard/okto-faucet) from the dashboard
    - Obtained your [API keys](/docs/developer-admin-dashboard/api-key) from the dashboard
    - Enabled the specific [Chains and Tokens](/docs/developer-admin-dashboard/wallet-control) you plan to use in your application
    - Optional: Enabled [Sponsorship](/docs/developer-admin-dashboard/sponsorship) for the desired chains

    If you need help, reach out to us on our <ExternalLink href="/docs/unity-sdk/troubleshooting">troubleshooting form</ExternalLink> and we will contact you.
    </CollapsibleCallout>

  - Okto API Keys: You need your `CLIENT_PRIVATE_KEY` and `CLIENT_SWA`. Obtain these from the [Okto Developer Dashboard](https://dashboard.okto.tech/login).
  - Okto Account Setup: Ensure you have sufficient <ExternalLink href="https://dashboard.okto.tech/okto-credits">Okto Credits</ExternalLink>, enable the required <ExternalLink href="https://dashboard.okto.tech/wallet-controls">Chains and Tokens </ExternalLink>, and optionally <ExternalLink href="https://dashboard.okto.tech/paymaster">Activate Sponsorship</ExternalLink>.
  - Google OAuth Credentials (if using Google authentication): Create OAuth 2.0 credentials in the <ExternalLink href="/docs/unity-sdk/authenticate-users/google-oauth/google-console-setup">Google Cloud Console</ExternalLink> to get your `GOOGLE_CLIENT_ID`.

  5. Start the Unity Project by running play.
