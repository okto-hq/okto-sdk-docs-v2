# JWT Authentication

Use Okto's Unity SDK to authenticate users with JWT tokens via the `Login` method. This method internally calls the Okto backend to verify the JWT and establish a secure session.

---

## Quick Overview

| Method       | What it does                                      |
|--------------|--------------------------------------------------|
| `Login(jwtToken)` | Verifies the JWT token and logs the user in. Returns true on success. |

---

## Example Code

```csharp
using OktoSDK.Auth;
using UnityEngine;
using System;

public class JwtAuthExample : MonoBehaviour
{
    public JwtAuthenticate jwtAuth;

    public async void LoginWithJwt(string jwtToken)
    {
        try
        {
            bool isSuccess = await jwtAuth.Login(jwtToken);
            if (isSuccess)
            {
                Debug.Log("Login successful!");
            }
            else
            {
                Debug.LogError("Login failed: Invalid JWT or session error.");
            }
        }
        catch (Exception ex)
        {
            Debug.LogError("Login failed: " + ex.Message);
        }
    }
}
