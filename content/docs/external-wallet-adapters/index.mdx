## Authentication Requirements

Before integrating any external wallet adapter, you'll need to set up authentication credentials:

- **Google OAuth credentials:** Create OAuth 2.0 credentials in the <ExternalLink href="https://console.cloud.google.com/">Google Cloud Console</ExternalLink> to get your `googleClientId`.

- **Okto Dashboard Setup:** You need your `clientPrivateKey` and `clientSWA`. Obtain these from the <ExternalLink href="https://dashboard.okto.tech/login">Okto Developer Dashboard</ExternalLink>.

## Available Adapters

### Wagmi

The Okto Wagmi Adapter enables integration of Okto's embedded smart contract wallets into applications built with the [Wagmi](https://wagmi.sh/) library of React hooks.

<Callout type="info">
**Perfect for:** dApps already using Wagmi that want to add social login wallet options.
</Callout>

### RainbowKit

For dApps using [RainbowKit](https://rainbowkit.com/) for wallet connection UIs, the Okto RainbowKit Adapter provides seamless integration through the familiar RainbowKit modal interface.

<Callout type="info">
**Perfect for:** dApps using RainbowKit that want to offer social login alongside traditional wallet options.
