The `getPortfolioActivity` function retrieves the user's portfolio activity history, including transactions, trades, and other relevant activities across all supported chains.

<ChainSupportStatus 
/>

### Example

<Accordions defaultValue="Usage">
<Accordion title="Usage">
<Tabs items={['TypeScript']}>
<Tab value="TypeScript">
```typescript
import { useOkto } from '@okto_web3/react-sdk';
import { getPortfolioActivity } from '@okto_web3/react-sdk'; // [!code highlight]

function PortfolioActivity() {
    const oktoClient = useOkto();

    async function fetchActivity() {
        try {
            const activity = await getPortfolioActivity(oktoClient); // [!code highlight]
            console.log('Portfolio activity:', activity);
        } catch (error) {
            console.error('Error fetching portfolio activity:', error);
        }
    }

    return (
        <button onClick={fetchActivity}>
            Fetch Portfolio Activity
        </button>
    );
}
```

## Get Portfolio Activity

<sub><i>async</i></sub> `getPortfolioActivity(oktoClient: OktoClient)` retrieves the user's portfolio activity history.

### Parameters

| Parameter | Type | Description | Required |
|-----------|------|-------------|-----------|
| `oktoClient` | `OktoClient` | Instance of OktoClient obtained from useOkto hook | Yes |

### Response

<Callout title="Success Response">

| Field Name | Type | Description |
|------------|------|-------------|
| `activities` | [`Promise<UserPortfolioActivity[]>`](/docs/react-sdk/technical-reference#models) | Array of user's portfolio activities |
