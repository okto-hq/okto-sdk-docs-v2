## What is Eliza?

<ExternalLink href="https://elizaos.github.io/eliza/">Eliza</ExternalLink> is an open-source framework designed for building AI agents that can interact across multiple platforms. It provides a modular and flexible architecture to create customizable, scalable, and stateful AI agents.


### Core Features of Eliza

- **Modular Design:** Uses plugins and services for easy customization
- **Scalable Knowledge:** Supports Retrieval-Augmented Generation (RAG)
- **Stateful Interactions:** Maintains context across conversations
- **Multi-Agent Support:** Runs multiple agents with unique configurations
- **Multi-Platform Support:** Integrates with Discord, Twitter, and more

### Eliza's architecture consists of:

- **Agent Runtime:** The central coordinator for message processing, memory, and actions.
- **Character Files:** Define agent personality, tone, and capabilities.
- **Clients:** Connect agents to platforms.
- **Actions:** Enable specific agent behaviors.
- **Evaluators:** Monitor and improve responses.
- **Providers:** Supply external data.
- **Memory & Knowledge Systems:** Store and retrieve conversation context and facts.
