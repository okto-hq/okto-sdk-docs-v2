<Callout title="What is Type*?">
**Type** refers to the family of chains or ecosystems a blockchain belongs to. For example, the EVM ecosystem includes all chains that run EVM-compatible nodes, sharing similar consensus mechanisms and transaction processing methods. Other ecosystems include Solana, Aptos, Bitcoin, and Cosmos, each with distinct approaches to transaction payload creation and processing. Understanding a chainâ€™s type is essential for constructing the appropriate payloads when interacting with that blockchain.
</Callout>
