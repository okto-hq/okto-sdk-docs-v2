## How to Create an Account

To create a new account for the dashboard:

1. Navigate to the dashboard [login page](https://dashboard.okto.tech).

2. Enter your preferred email address to log in and verify it using the OTP sent to your inbox.

<div className="flex space-x-2 justify-center">

    ![Dashboard Login Screen](/images/dashboard-img-v2/dashboard-login-screen-v2.png)

    ![OTP Verification Screen](/images/dashboard-img-v2/dashboard-otp-screen-v2.png)

</div>

3. If you are creating an account for the first time, you will be prompted to enter you project name and initialise yourself as a client on Okto by deploying the Client Smart Wallet Account (ClientSWA) on the Okto chain. 
Okto will deploy this contract for you for free. Simply click continue to proceed.

<div className="flex space-x-2">

    ![Create Project](/images/dashboard-img-v2/dashboard-login-project-v2.png)

    ![Creat Client](/images/dashboard-img-v2/dashboard-login-screen-1-v2.png)

    ![Client Creation Success](/images/dashboard-img-v2/dashboard-login-screen.png)

</div>

4. Now, you are logged in to your new dashboard account.

![OTP Verification Screen](/images/dashboard-img-v2/dashboard-home-screen-v2.png)


<Callout title="Note">
Currently, only one default project is associated with one email address.
</Callout>
